# Python - Async Comprehension

This project covers asynchronous generators and async comprehensions in Python 3.

## Learning Objectives

- How to write an asynchronous generator
- How to use async comprehensions
- How to type-annotate generators

## Requirements

- Python 3.8 (Ubuntu 20.04 LTS)
- pycodestyle 2.5.x

## Tasks

### 0. Async Generator
`0-async_generator.py` — Coroutine that loops 10 times, waits 1 second asynchronously each time, and yields a random float between 0 and 10.
